# RemOnline Vue frontend

Минималистичный одностраничный интерфейс на Vue 3 поверх существующего FastAPI‑бэкенда.

## Стек

- Vue 3 (Composition API, без роутера)
- Vite
- Axios

## Структура

- `index.html` — корневой HTML для Vite
- `src/main.js` — инициализация Vue‑приложения
- `src/App.vue` — основное приложение: логин, список клиентов и заказов
- `src/services/api.js` — тонкий клиент к REST API (`/api/...`)
- `src/style.css` — минималистичные стили (тёмная тема)

## Запуск

1. Установите Node.js (LTS) и npm / pnpm / yarn.
2. В корне фронтенда установите зависимости:

```bash
cd frontend
npm install
```

3. Запустите dev‑сервер:

```bash
npm run dev
```

По умолчанию Vite поднимается на `http://localhost:5173`.

## Интеграция с бэкендом

- Бэкенд FastAPI ожидается по адресу `http://localhost:8000`.
- Vite настроен проксировать запросы `/api/**` на `http://localhost:8000`, поэтому в браузере все запросы идут на тот же origin (`5173`), и CORS‑настройки бэкенда можно не трогать для локальной разработки.
- Авторизация:
  - `POST /api/auth/login` — вход по email/паролю, ответ содержит `access_token` и `refresh_token`.
  - Токены сохраняются в `localStorage` и автоматически подставляются в заголовок `Authorization: Bearer ...`.
  - При 401 фронтенд один раз пробует обновить токен через `POST /api/auth/refresh` и повторить запрос.

## Что уже реализовано

- Форма логина и выход из системы.
- Таблица клиентов на основе ответа `GET /api/client/all`.
- Таблица заказов на основе ответа `GET /api/order/all`.
- Простая индикация загрузки и ошибок.
